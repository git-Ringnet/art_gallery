   ✅ Quên mật khẩu
   ✅ Đổi mật khẩu
   ✅ Quản lý hồ sơ
   ✅ Đăng xuất
   ✅ Bảo vệ tất cả routes

📚 TÀI LIỆU:
   - AUTH_SETUP.md (Chi tiết)
   - INSTALLATION_SUMMARY.md (Tóm tắt)

   - Tất cả routes đã được bảo vệ bằng middleware 'auth'
   - Dashboard và các trang khác giữ nguyên
   - Layout app.blade.php hiển thị user thật
   - Đổi mật khẩu ngay sau lần đăng nhập đầu

💡 TẠO USER MỚI:
   php artisan tinker
   >>> User::create([
       'name' => 'Tên',
       'email' => 'email@example.com',
       'password' => Hash::make('password'),
       'email_verified_at' => now()
   ]);
